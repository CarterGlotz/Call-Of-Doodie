# Brain

## Mental model

Call of Doodie wins by delivering a joke every few seconds without sacrificing
mechanical trust. The player should feel that the game is silly on purpose, not
sloppy by accident.

## Working heuristics

- If a feature is funny but unreadable, simplify it until the joke survives
- If a feature is deep but joyless, it probably does not belong here
- New content should create fresh run stories, not just bigger numbers
- Mobile convenience should not quietly damage the core arcade skill loop
- Anything that improves sharing, rivalry, or repeat sessions has outsized value

## Current strategic beliefs

- The strongest differentiation is comedy plus polished moment-to-moment feel
- Shareability and challenge links are a major growth lever
- Daily challenges, ghost competition, and seeded runs can become a studio-grade moat
- The project is a strong template for other humor-first VaultSpark arcade products

## What must stay true while scaling

- Short session accessibility
- Strong feedback for weapons, enemies, and perks
- Clear run-state communication even as systems multiply
- Original VaultSpark personality over generic genre imitation
