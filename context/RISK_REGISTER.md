# Risk Register

### Readability collapse at higher content density

- Type: design
- Likelihood: medium
- Impact: high
- Mitigation: preserve strong telegraphs, visual hierarchy, and HUD clarity
- Trigger to watch: playtests where players cannot explain what killed them
- Owner: design

### Parody identity drifting into trademark-sensitive imitation

- Type: brand and IP
- Likelihood: medium
- Impact: high
- Mitigation: keep brand system, provenance notes, and creative direction records current
- Trigger to watch: direct copying of third-party assets, phrases, or military aesthetics
- Owner: creative direction

### Live competition features outgrowing current backend setup

- Type: technical
- Likelihood: medium
- Impact: medium
- Mitigation: sequence Supabase improvements before adding heavier competitive features
- Trigger to watch: leaderboard abuse, identity conflicts, or feature requests needing stronger server authority
- Owner: engineering

### App.jsx complexity slowing future iteration

- Type: technical
- Likelihood: high
- Impact: medium
- Mitigation: continue careful extraction around the game loop without destabilizing the ref-driven architecture
- Trigger to watch: bug-fix speed dropping or feature work requiring repeated changes in the same fragile regions
- Owner: engineering
