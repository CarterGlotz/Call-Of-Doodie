# Innovation Pipeline

## Innovation thesis

- What kind of innovation this project should pursue: comedic, social, highly shareable systems that deepen replayability without requiring heavy backend complexity
- What kind of innovation would be off-brand: grim realism, system bloat that slows the arcade loop, or trend-chasing features with no comedic payoff
- What market or genre complacency we want to defeat: shooters that feel mechanically competent but socially forgettable and tonally generic
